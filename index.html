== JK Drive Amplifier ==

This plugin is a mono LV2 audio plugin. It has two controls, drive and output gain.

The plugin does basic wave shaping distortion with 2x oversampling. It produces both even and odd order hamonic distortion.

Use it to add a little brightness or to add some saturation effects.
